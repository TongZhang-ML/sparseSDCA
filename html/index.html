Package: Acclerated Proximal Stochastic Dual Coordinate Ascent for Solving L1-L2 Regularized Binary Linear Classification & Regression

Introduction

This code employs acclerated prox-SDCA to train binary linear optimization problems based on the paper Accelerated Proximal Stochastic Dual Coordinate Ascent for Regularized Loss Minimization.

It may be used to do experiments similar to what's reported in the paper (with different default parameter configurations)

Installation

go to src and type make

Experiments

use the "exp/run" shell script to do experiments
